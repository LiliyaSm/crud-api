## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/LiliyaSm/crud-api.git
cd crud-api
```

```bash
npm install
```

To run the app in development mode:

```bash
npm run start:dev
```

To run the app in production mode:

```bash
npm run start:prod
```
